The Zen of Angular

Angular is built around the belief that declarative code is better than imperative when it comes to building UIs and wiring software components together, while imperative code is excellent for expressing business logic.

$rootScope

$rootScope & $scope

.controller("tasteController", function ($rootScope, $scope) {
  $rootScope.name = 'Leo';
  $scope.location = 'San Francisco';
});

Every application has a single root scope.

All other scopes are descendant scopes of the root scope.

$scope is an object that refers to the application model.

$digest vs $apply

.controller("tasteController", function ($scope) {
  $scope.$digest();
});

.controller("tasteController", function ($scope) {
  $scope.$apply();
});

$watch

Scopes provide APIs ($watch) to observe model mutations.

.controller("tasteController", function ($scope) {
  $scope.$watch('name', function(newValue, oldValue) {
    $scope.myCallback(newValue);
  });
});

      templateUrl: 'templates/home.html',

    otherwise({
      redirectTo: '/home'
    });
});
